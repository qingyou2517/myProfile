项目经历

Every Note 云笔记

源码链接 项目预览

项目背景：模拟日常学习笔记的需求，旨在提供一个便捷的在线笔记服务，用户可以在线创建、编辑、保存和分享笔记，并且可以在任何 pc 设备上访问和编辑笔记。

技术选型：使用 less、 Vue2、Vue-Router、 Vue-Cli、 Vuex、Element-UI、 ES6、node.js、Axios、 webpack 等技术来构建前端界面，实现路由和脚手架、管理状态、构建UI组件、编写JavaScript代码、构建后端服务、实现前后端数据交互以及构建和打包项目。

技术难点：比如如何实现前后端数据交互，如何确定当前点击的是哪条笔记等。为了解决这些问题，我对axios二次封装来实现自动切换开发、生产环境与后台交互；并设计出借助路由守卫从 $rotue.query.noteId 获取当前 noteId，通过组件通信确定处于哪一个笔记本的方案。

项目亮点：为提升代码复用率，项目中为许多功能函数做了抽象封装；为优化对图片的加载性能，使用了懒加载，把单页面应用的js根据路由来打包；为优化用户的体验，切换注册/登录时添加了缓和的动画效果，笔记的内容也支持 markdown 语法 ，笔记(本)为空时引导用户操作等，后续还会添加更多优化。

项目收获：最终，我成功地完成了该项目，并更加深入地了解了前端技术。我把在接口封装中使用 Promise 封装 axios 的经验总结成了一篇博客《前后端分离项目中如何封装 axios》。同样，也把完整的开发过程、bug 处理总结成了一组博客《云笔记开发思路》。

React 大屏可视化

源码链接 项目预览

模仿素材网站开发的基于 React / Echarts 的大屏可视化动态图表项目，项目中大量使用了 Grid & flex 布局。
