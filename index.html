React 大屏可视化

源码链接 项目预览

模仿素材网站开发的基于 React / Echarts 的大屏可视化动态图表项目，项目中大量使用了 Grid & flex 布局。

Every Note 云笔记

源码链接 项目预览

模拟日常学习笔记的需求，开发的一款简洁好用、可后台交互、前后端分离的 PC 端笔记项目。

主要技术栈：less / Vue2 / Vue-Router / Vue-Cli / Vuex / Element-UI / ES6 / node.js / Axios / webpack

从产品需求入手，功能涵盖 注册登录、登录跳转、展示头像&笔记详情、笔记本切换、垃圾站回收等。

为提升代码复用率，项目中为许多功能函数做了抽象封装；

为优化对图片的加载性能，使用了懒加载；

为优化用户的体验，切换注册/登录时添加了缓和的动画效果，笔记的内容也支持 markdown 语法 ，笔记(本)为空时引导用户操作等，后续还会添加更多优化。

我把在接口封装中使用 Promise 封装 axios 的经验总结成了一篇博客《前后端分离项目中如何封装 axios》。
